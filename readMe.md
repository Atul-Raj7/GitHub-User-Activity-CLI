<h1 align=center>GitHub-Task-Tracker-CLI</h1>

Nothing much, just a humble cli tool to take a quick look at your GitHub activity.

---

### Requirements 

- NodeJS - 22.19.0
- A GitHub Account (Obviously) with some activity in it
---

### Installation

Download the code in your desired folder/workspace 

---

### Run the tool

```bash
node index.java <your_github_username>
```

>Note: <br> 
<br> 
<your_github_username> = Your GitHub Username
<br> 
<br> 
Example: If your Username is "Mota-Aadmi787", then:
<br>
<br>
node index.js Mota-Aadmi787
